# Conversation Backup 설치 가이드

Claude Code에서 `/compact` 또는 auto-compact 실행 시 대화 내용을 자동으로 저장하는 스킬입니다.

## 설치 방법

### 방법 1: tarball 사용 (권장)

```bash
# 1. tarball 압축 해제
mkdir -p ~/.claude/skills
tar xzf conversation-backup.tar.gz -C ~/.claude/skills/

# 2. 실행 권한 부여
chmod +x ~/.claude/skills/conversation-backup/backup.sh

# 3. Claude Code 재시작
```

### 방법 2: 수동 설치

```bash
# 1. 폴더 생성
mkdir -p ~/.claude/skills/conversation-backup

# 2. 파일 복사
cp backup.sh ~/.claude/skills/conversation-backup/
cp convert_inline.py ~/.claude/skills/conversation-backup/
cp SKILL.md ~/.claude/skills/conversation-backup/

# 3. 실행 권한 부여
chmod +x ~/.claude/skills/conversation-backup/backup.sh
```

### Hook 설정

`~/.claude/settings.json` 파일에 다음 내용을 추가:

```json
{
  "hooks": {
    "PreCompact": [
      {
        "matcher": "*",
        "hooks": [
          {
            "type": "command",
            "command": "bash ~/.claude/skills/conversation-backup/backup.sh",
            "timeout": 30
          }
        ]
      }
    ]
  }
}
```

이미 `hooks` 섹션이 있다면 `PreCompact` 부분만 추가하세요.

### Claude Code 재시작

설정 변경 후 Claude Code를 재시작해야 hook이 적용됩니다.

## 사용법

자동으로 작동합니다:
- `/compact` 명령 실행 시
- auto-compact 트리거 시

대화 내용이 프로젝트의 `.conversation/` 폴더에 저장됩니다.

## 출력 파일

```
{프로젝트}/.conversation/
├── session-abc123.md    # 세션 abc123
├── session-def456.md    # 세션 def456
└── ...
```

## 기능

- **같은 세션 = 같은 파일**: 여러 번 compact 해도 내용이 추가됨
- **읽기 쉬운 마크다운**: User/Assistant 메시지 구분
- **자동 정리**: 최근 100개 파일만 유지
- **세션 추적**: 각 세션별로 별도 파일

## 요구사항

- Python 3.x
- Bash

## 제거 방법

```bash
# 1. settings.json에서 PreCompact hook 제거
# 2. 스킬 폴더 삭제
rm -rf ~/.claude/skills/conversation-backup
# 3. Claude Code 재시작
```
